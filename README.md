# Nike Air Zoom - Landing Page

Projeto desenvolvido com foco em responsividade e interação básica utilizando JavaScript.

## 🚀 Funcionalidades

* Layout responsivo (mobile → desktop)
* Troca dinâmica de cor e imagem
* Animação na troca do produto

## 🛠 Tecnologias

* HTML5
* CSS3
* JavaScript

## 📱 Responsividade

Adaptado para:

* Mobile
* Tablet
* Desktop
